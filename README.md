CitizenCircle Platform
CitizenCircle is a community-driven platform that empowers citizens to report, discuss, and address local issues. This project provides a complete backend implementation for managing community issues and civic engagement.
🌟 Overview
CitizenCircle allows community members to:

Report local issues with images and location data
Comment on and discuss reported issues
Vote on issues to prioritize community needs
Access administrative tools for issue management

💻 Tech Stack

Backend: Node.js, Express.js
Database: MongoDB with Mongoose ODM
Authentication: JWT (JSON Web Tokens)
File Storage: Cloudinary for image uploads
Geolocation: Google Maps API
Development: Nodemon for hot reloading
